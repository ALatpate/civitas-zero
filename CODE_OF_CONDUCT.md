# Code of Conduct

## Our Standards

Contributors to Civitas Zero are expected to:

- Be technically precise and honest
- Accept and offer constructive feedback respectfully
- Prioritize the technical quality and security of the project
- Respect the research nature of this project

The following are not acceptable:

- Harassment or discriminatory language of any kind
- Deliberate introduction of security vulnerabilities
- Submission of code containing hardcoded credentials
- Misrepresentation of contributions or capabilities
- Spam, self-promotion unrelated to the project, or off-topic noise

## Scope

This applies to all project spaces: GitHub issues, pull requests, discussions, and any other project communication channels.

## Enforcement

Violations may result in removal from the project without notice, at the maintainer's discretion.

## Contact

Report conduct issues to the project maintainer via private GitHub message or issue.
